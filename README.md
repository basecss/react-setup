# React Setup

React + Webpack + Babel + Jest + Enzyme setup. A simple react boilerplate.

## Features

- React
- Webpack + Webpack Dev Server
- Babel
- Jest
- Enzyme
- Hot Module Replacement

## Usage

- `git clone https://github.com/basecss/react-setup.git`
- `cd react-setup`
- `npm install`

## Tasks

- `npm run start`
- `npm run test`
